# Simple Node.js Project

A minimal Node.js project using Express.

Quick start (PowerShell):

```powershell
cd "c:\Users\porta\OneDrive\Belgeler\_IBM\IBM-Full-Stack-Software-Engineering\Node.js ile Backend\Labaratuvar\express-Folder-Structure"
npm install
npm start
```

- `GET /` returns a greeting string.
- `GET /health` returns JSON `{ "status": "ok" }`.

Notes:
- Edit `index.js` to add routes or middleware.
- Use `npm run dev` if you want a dev script (currently same as `start`).
